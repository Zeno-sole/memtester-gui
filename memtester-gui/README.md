# Memtester GUI - 内存测试工具

基于 Qt6 开发的 Memtester 图形界面，采用 iOS 风格设计。

## 功能特性

- **iOS 风格设计**：圆角卡片、系统配色、清晰的层次结构
- **完整 CLI 功能**：支持所有 memtester 命令行选项
- **测试选择**：可选择要运行的内存测试项目
- **内存设置**：配置内存大小（B/KB/MB/GB）和循环次数
- **高级选项**：物理地址映射、解锁内存模式
- **实时进度**：实时测试结果与可视化进度指示器
- **详细日志**：与主题协调的日志输出窗口

## 界面颜色方案

- **主背景**：#F2F2F7（iOS 系统灰色背景）
- **卡片背景**：#FFFFFF（白色）
- **边框**：#E5E5EA（浅灰色）
- **主色调**：#007AFF（iOS 蓝色）
- **成功色**：#34C759（iOS 绿色）
- **警告色**：#FF3B30（iOS 红色）
- **文字色**：#000000（黑色）/ #3C3C43（次级文字）
- **日志区域**：白色背景 + 深色文字（与主题协调）

## 系统要求

- Qt6（Core、Gui、Widgets）
- CMake 3.16+
- C++17 编译器
- Linux/Unix 系统

## 编译安装

```bash
mkdir build
cd build
cmake ..
make -j$(nproc)
```

## 使用方法

```bash
./memtester-gui
```

### 设置选项

1. **内存大小**：要测试的内存量（例如 100 MB）
2. **循环次数**：测试迭代次数（0 = 无限循环）
3. **测试选择**：选择要运行的内存测试项目
4. **高级选项**：
   - 物理地址：映射特定物理内存地址
   - 不解锁内存：跳过 mlock 以获得更快但可靠性较低的测试
